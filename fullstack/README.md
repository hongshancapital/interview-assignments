### Development

```bash
$ cd fullstack
$ yarn install
$ npm run dev  
$ open http://localhost:3000/

$ npm run dev  前后端调式模式. 第一次使用可能需要多等会,生成前端,后端打包文件.有报错不急.
$ npm run test 跑覆盖率
```
