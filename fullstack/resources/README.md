# Short-link web

## Struct
`create-react-app + typescript + jest`

## init
```shell
# if node_modules not installed
npm install
```

## dev
```
npm run start
```

## build
```
npm run build
```

- if you want to test the build file, please start a http-server
```shell
# eg: MacOS
cd build && PHP -S localhost:3000
```

## test
```
npm run test
```
