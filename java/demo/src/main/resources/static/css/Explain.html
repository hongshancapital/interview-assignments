竞猜榜单说明

1.使用钻石参与竞猜游戏；

2.单次竞猜最低1000钻石，最高上限9900000钻石；

3.日榜单每日一期，比赛结束时间在当日12：00至次日11：59的比赛计入当日榜单；

4.周榜单每周一期，比赛结束时间在本周一12：00至次周一11：59的比赛计入本周榜单；

5.根据用户的盈利钻石数和亏损钻石数设置红单榜和天台榜；

6.每日天台榜第一名可获得不超过100万钻石的返奖，每周天台榜可获得不超过500万钻石的返奖（返奖可提现）；

7.奖励将在48小时内发放（节假日顺延）；

8.对于恶意刷榜、作弊等行为，一经发现，永久冻结账号。

声明：

1.本应用社区竞猜活动仅作为用户娱乐用途，任何人不得用于非法用途，否则责任自负。用户竞猜数据均为用户个人意见之表达，与本应用无关

2.竞猜数据仅提供中国体育彩票参考使用，不涉及任何购彩业务
