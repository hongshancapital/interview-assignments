短路径设计文档
=

* 存储

>jvm内部两个ConcurrentHashMap, 第一个map的key是长路径value是短路径，第二个map的key是长路径value是短路径，这样做数据会有冗余但是正反两种查询都比较快

* 接口1 由长路径获取短路径

>如果记录已经存在， 直接返回对应的短路径

>如果记录不存在，随机生成不冲突的短路径， 然后保存信息，再返回短路径

* 接口2 由短路径获取长路径

>如果记录已经存在， 返回长路径

>如果记录不存在， 返回空
