# Carousel 组件说明

## 编码风格
- 统一 eslint 风格
- 函数式，尽可能抽离副作用
- hooks 状态共享

## 组件功能
- 轮播功能
- 支持时间滑块停留时间配置
- 支持设置默认时间
- 默认模块设置

## 模块拆分
- Carousel 组件
- CarouselDot 组件

## 涉及hook
- useEvent 处理闭包和和缓存函数引用
- useCarouselAnimate 抽离 Carousel 状态
