## 感受

1. hooks 整体上确实偏声明式/函数式风格，有函数式组件的轻量又能有代数效应，在一些逻辑封装上确实能做到高内聚低耦合
2. 一开始设计的时候并没有尝试写 hooks（之前只是有概念，也知道底层原理，但是没有实践），所以在结合封装 Class 时有一点后悔如此设计，好在找到套路后好像还行
