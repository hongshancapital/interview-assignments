## 轮播组件

### API

### Carousel

| 参数      | 说明                      | 类型    | 默认值 |
| --------- | ------------------------- | ------- | ------ |
| autoplay  | 是否自动播放              | boolean | false  |
| className | 自定义类名                | string  | -      |
| duration  | 自动播放延时时间，单位 ms | number  | 3000   |
| showArrow | 是否显示左右切换箭头      | boolean | false  |
