# 短域名服务

### 实现API接口

* 短域名存储接口：接受长域名信息，返回短域名信息。
* 短域名读取接口：接受短域名信息，返回长域名信息。
* 限制：短域名长度最大为8个字符

### 技术要点

* 框架:Spring Boot；
* 文档:Swagger API； 
* 访问路径：
* swagger2接口说明:
     [http://地址:端口/应用名/swagger-ui.html](http://127.0.0.1:8080/swagger-ui.html)
   
* swagger2接口源信息:
     [http://地址:端口/应用名/v2/api-docs](http://127.0.0.1:8080/v2/api-docs)

* 单元测试:JUnit;
* 测试报告:Jacoco(提交截图)；
* 映射数据:存储在JVM内存，防止内存溢出；
